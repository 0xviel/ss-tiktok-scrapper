# SS TIKTOK SCRAPPER

## Instalasi
```bash
# Clone this repository
$ git clone https://github.com/0xviel/ss-tiktok-scrapper
# Go into the repository
$ cd ss-tiktok-scrapper
# Install Modul Go
$ go mod init tiktok-scraper
$ go mod tidy
# Run the program 
$ go run ssstiktok.go <tiktok_url>
```
